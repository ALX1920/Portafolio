errores clásicos / classic errors  
========================================== _/

Tuve problemas al cerrar head, al final entendí que tenía que agregar un (/).
I had problems closing the head tag. In the end, I understood that I had to add a (/).

<head>
</head>

/_ ==========================================
falta de etiqueta / missing tag  
========================================== _/

No sabía que para agregar estilos también se requerían etiquetas. A la hora de querer realizar un cambio en algo en específico, este se cambiaba todo. Al principio me frustré, pero al investigar aprendí que las etiquetas me podían facilitar los cambios de estilos, así que empecé a hacer uso de ellas.

I didn’t know that to add styles you also needed tags. When I wanted to make a change to something specific, everything would change. At first I got frustrated, but after researching, I learned that tags could make style changes easier, so I started using them.

Aunque también tuve problemas al principio de usarlas, ya que, a la hora de llamarlas, estas no realizaban los cambios. Pero después de un rato me enteré de que para llamarlas se necesita colocar un punto (.) antes del nombre de la etiqueta:

However, I also had problems at the beginning when using them, because when I tried to call them, they didn’t apply the changes. But after a while, I found out that to call them you need to put a dot (.) before the tag name:

.(nombre)

Después de aprender esto, aun así tuve errores, ya que olvidaba poner el punto, pero mi frustración ya no era igual.
After learning this, I still made mistakes because I forgot to add the dot, but my frustration wasn’t the same anymore.
